# AI Usage in SBOMVert

## Overview

SBOMVert is primarily developed through manual engineering based on the [research](https://arxiv.org/abs/2510.05798) described in the README.

Artificial Intelligence (AI) tools are **not used to generate core logic, security-relevant functionality, or SBOM data processing components**.

AI is used in a limited and controlled manner to assist with specific development tasks, as detailed below.

---

## Scope of AI Usage

AI assistance has been used in the following areas:

* **User Interface (UI) development**

  * Prototyping frontend components
  * Generating initial layout or visualization ideas
  * Suggesting improvements to usability and presentation

* **Non-critical code scaffolding**

  * Drafting boilerplate code that is subsequently reviewed and modified

All AI-generated outputs are **systematically reviewed, edited, and validated** before inclusion in the project.

---

## Licensing and Compliance

All contributions, including those assisted by AI, comply with the **Apache License 2.0**.
